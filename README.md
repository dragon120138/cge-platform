# 中国42部门CGE政策模拟平台 v2.0

> **China 42-Sector CGE Policy Simulation Platform** — 基于 Johansen 对数线性化方法的可计算一般均衡模型，覆盖中国 42 个国民经济部门，支持 12 季度（3 年）动态政策模拟。

---

## 一、项目意图

**一句话：做一个能让非经济学家也能用的宏观经济政策沙盘。**

传统 CGE 模型依赖 GAMS/GEMPACK 等商业求解器，年费动辄数万美元，且操作门槛极高。本项目用纯 Python + NumPy 实现了 Johansen 对数线性化求解器，零商业依赖，30 秒内完成一次完整的政策冲击模拟。

你可以在网页上拖滑杆设定政策参数（减税、降息、定向投资、TFP 提升等），模型实时计算该政策对中国 GDP、就业、物价、42 个行业产出、财政平衡和三部门信心指数的影响，并生成 12 季度的动态过渡路径。

**目标用户**：经济学研究者、政策分析师、财经类学生、对宏观政策模拟感兴趣的技术爱好者。

**核心数据流**：

1. **SAM 构建** (`data/data_pipeline.py`)：基于 2017 年 42 部门 IO 表 + 2025 年宏观统计校准，生成基准 SAM
2. **参数校准** (`cge_core/calibrator.py`)：从 SAM 反推份额参数和规模参数
3. **Johansen 求解** (`cge_core/johansen.py`)：将非线性均衡系统对数线性化，转化为线性方程组，毫秒级求解
4. **动态模拟** (`cge_core/johansen.py` 中的 `DynamicJohansenSolver`)：递归动态 + AR(1) 衰减路径，12 期
5. **场景管理** (`cge_core/scenario.py`)：基线 ↔ 反事实对比，自动计算偏离度
6. **报告生成** (`dashboard/report_generator.py`)：AI 分析（需 GLM API）或模板降级

## 三、核心功能

### 政策工具箱

| 类别 | 工具 | 说明 |
|------|------|------|
| 财政政策 | 消费税、企业所得税、生产税、政府支出 | 税率/支出幅度可调，支持 ± 范围 |
| 货币政策 | 利率（LPR）| ±200 基点，模拟加息/降息 |
| 结构性政策 | 定向产业投资 | 42 部门任选（支持多选），±50% 投资变动 |
| 结构性政策 | 全要素生产率（TFP）冲击 | 全行业或定向行业，模拟"新质生产力"提升 |
| 时序控制 | 永久实施 / 一次性脉冲 / 预告后实施 | 三种政策实施路径 |

### 输出维度

- **宏观**：GDP、CPI、就业、居民福利（EV）、瓦尔拉斯残差
- **财政**：基准赤字、政策对财政平衡的影响、12 季度财政盈余/赤字动态路径
- **信心指数**：消费者、企业、投资者三部门信心变化（静态 + 12 季度动态）
- **行业**：42 部门产出/价格/消费变化排名、点击任一行业查看 12 季度产出动态曲线
- **报告**：AI 生成（需 GLM API Key）或模板生成两类报告——宏观分析 + 产业分析

### 模型规格

| 项目 | 规格 |
|------|------|
| 求解方法 | Johansen 对数线性化（纯 NumPy，无需 GAMS/Pyomo） |
| 部门分类 | 国家统计局 2017 年 42 部门投入产出分类 |
| 基准年 | 2025 年（GDP 140.19 万亿元，财政赤字 7.13 万亿，LPR 3.00%） |
| 动态期限 | 12 个季度（3 年），含中期效应分析 |
| 宏观闭合 | 凯恩斯模式（就业可变）/ 新古典模式（充分就业）可选 |
| 需求系统 | 线性支出系统（LES），Frisch = -1.5 |
| 生产结构 | 嵌套 CES（增加值 CES + 中间投入 Leontief） |
| 贸易 | 小国开放经济，Armington/CET 弹性部门异质 |
| 求解性能 | 静态 <120ms，动态 <110ms |

## 六、设计决策 FYI

### 6.1 TFP 的外生化设定

**问题**：在标准增长模型中，全要素生产率（TFP）是内生变量——它由技术进步、制度改进、资源配置效率等因素共同决定，政策无法直接"设定"TFP。

**本项目中的处理**：TFP 被设定为**可外生调整的政策变量**。理由如下：

1. **模拟目的优先**：本项目的核心目的是评估"新质生产力建设在不同前景下发挥的效应"。如果将 TFP 视为内生，则无法分离政策效应——我们需要一个可控的 TFP 旋钮来回答"如果 TFP 提升 1%，会对经济产生多大影响"这样的反事实问题
2. **实际可操作化**：虽然 TFP 不能被直接拨动，但现实中存在大量间接干预渠道——R&D 税收抵免、国家实验室投入、产学研补贴、人力资本投资等。TFP 参数可以理解为这些政策投入的**约简代理变量**
3. **敏感性分析框架**：将 TFP 外生化后，用户可以测试不同强度的"新质生产力冲击"（如 +1% / +3% / +5%），观察其对 GDP、就业、行业结构的差异化影响——这本质上是情景分析而非预测
4. **学术先例**：Dixon & Rimmer (2002) 在 MONASH 模型中将技术变量作为外生冲击纳入闭合；世界银行的 LINKAGE 模型同样将 TFP 设为情景外生。这不是 hack，是成熟的建模范式

**窄范围设定**：TFP 冲击幅度控制在 ±5% 以内，因为 (a) 年化 TFP 增长率的历史波动本身极小（中国近年约 0.5-2%）；(b) Johansen 方法在大冲击下的线性化误差会增大

### 6.2 定向产业投资支持的设定逻辑

**问题**：现实中的产业政策不是给一个行业"一刀切"拨款。政策有力度差异（补贴 vs 低息贷款 vs 税收返还 vs 直接采购），有时间结构（实施前预览、逐步推出、退出机制），且不同行业的投资乘数不同。

**本项目中的处理**：

1. **多行业同时支持**：点击「+ 添加行业」按钮可以添加任意多行定向投资，每行独立设定行业和投资增幅幅度（-50% ~ +50%）
2. **三种实施时序**：
   - **永久实施**：投资在 t=0 立即到位并持续 12 个季度，模拟长期产业政策
   - **一次性脉冲**：仅在 t=0 投入，后续回归基准路径，模拟短期刺激（如"设备更新补贴"）
   - **预告后实施**：政策在 t=0 宣布，t=N 实施——居民和企业可以提前调整行为（前瞻效应），例如新建产能需要前期筹备
3. **财政负担内化**：定向投资的财政成本（假设补贴率 30%）计入政府预算约束，影响财政盈余/赤字动态，避免"免费午餐"幻觉
4. **行业间溢出**：投资增加→对上游原材料行业产生中间投入需求（Leontief 结构）→对下游行业降低价格（成本传导）→通过居民收入再传导至消费品行业——全套一般均衡链条

**重要说明**：定向投资是一个**高度简化的建模**。现实中的产业政策包括准入管制、国产替代要求、技术标准制定、人才培养等非财政工具，这些在本模型中尚未建模。欢迎社区补充。

## 八、技术说明

### 求解方法

Johansen 对数线性化：将非线性一般均衡系统在基准均衡点附近做对数线性化，转化为 `Ax=b` 线性方程组求解。优势：速度快（毫秒级）、数值稳定、无需商业求解器。局限：大冲击（>10%）下线性化误差增大。

### 信心指数

三部门信心指数基于多因子加权：

- **消费者信心** = 3.0×GDP + (-2.0)×CPI + 1.5×就业 + 0.5×财政 + 3.0×TFP
- **企业信心** = 2.5×GDP + (-3.0)×利率 + 1.0×投资(clip±10%) + 0.8×财政 + 10.0×TFP
- **投资者信心** = 2.0×GDP + (-2.5)×利率 + 1.0×汇率 + 1.0×财政 + 5.0×TFP

动态路径采用 AR(1) 衰减（持续性参数 0.85/季度）。

### 局限性

- 基准 SAM 为基于公开数据的校准值，非精确统计口径
- Johansen 方法对大冲击（>10%）的线性化误差会增大
- TFP 在模型中为外生调整（见 §6.1 说明）
- 不含银行系统/金融加速器（`financial_accelerator.py` 为基础框架，尚未集成到主求解器）
- 信心指数为简化因子模型，非结构化的预期形成

## 九、欢迎尝试 & 进一步优化

这个项目是一个**快速原型**——能跑、能看、能用，但绝对不完美。如果你感兴趣，以下方向值得推动：

1. **数据升级**：接入 2020 年或更新的 IO 表，替换当前基于 2017 年表的校准
2. **金融部门建模**：加入银行资产负债表、信贷渠道、影子银行、货币市场
3. **TFP 内生化的部分尝试**：将 R&D 投入→知识资本→TFP 链条建模（半内生增长）
